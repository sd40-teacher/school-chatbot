# 📁 학교 자료 폴더

이 폴더에 학교 소개 PDF 파일들을 업로드하세요.

## 📄 업로드할 PDF 파일

선생님께서 준비하신 **학교소개.pdf** 파일을 이 폴더에 업로드해주세요.

### 추가 가능한 파일들:
- 교육과정.pdf
- 입학안내.pdf
- 진로진학.pdf
- 학교생활.pdf
- 동아리소개.pdf
- 기타 학교 관련 자료

## 📌 주의사항

1. **PDF 형식만 지원**됩니다
2. 파일명은 **한글 가능**합니다
3. 파일 개수는 **제한 없음**
4. 파일 크기는 각 **50MB 이하** 권장

## 🔄 파일 업데이트 방법

### 매년 자료 업데이트 시:
1. 이 폴더(`data/school_docs/`)로 이동
2. 기존 PDF 파일 삭제
3. 새로운 PDF 파일 업로드
4. Commit changes 클릭
5. Streamlit이 자동으로 재배포 (2-3분 소요)
6. 완료!
